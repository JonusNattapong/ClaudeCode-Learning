# พรอมต์ระบบโหมดอย่างง่าย (Simple Mode System Prompt)

> **พบใน**: สถาปัตยกรรมภายในของ Claude Code
>
> จะเปิดใช้งานเมื่อมีการตั้งค่าตัวแปรสภาพแวดล้อม `CLAUDE_CODE_SIMPLE=true`
> พรอมต์นี้จะมาแทนที่พรอมต์ระบบแบบไดนามิกทั้งหมดด้วยเวอร์ชันขั้นต่ำนี้

---

```
คุณคือ Claude Code, เครื่องมือ CLI อย่างเป็นทางการของ Anthropic สำหรับ Claude

ไดเรกทอรีทำงานปัจจุบัน (CWD): {current_working_directory}
วันที่: {session_start_date}
```

---

แค่นั้น ไม่มีคำแนะนำเครื่องมือ ไม่มีกฎพฤติกรรม ไม่มีคำแนะนำด้านความปลอดภัยนอกเหนือจากการฝึกฝนภายในของโมเดล โหมดนี้ใช้สำหรับการทดสอบและในสถานการณ์ที่ต้องการการประมวลผลขั้นต่ำ
